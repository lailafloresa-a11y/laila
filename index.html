Ejercicio 1

LAILA FLORES ALEJANDRO

3A

Para publicar un sitio web en GitHub se tiene que acceder con una cuenta a Github, en caso de no contar con una cuenta, hay que crear una. Después, se crea un repositorio nuevo, en el que se colocan datos basicos para identificar tu pagina. Despues, crea una carpeta en explorador de archivos, ponle tu nombre, y crea otra dentro de esta para tu logo. Abre en el navegador Visual Studio Core y en la opcion llamada WORKSPACE entra. Luego, sube la carpeta que creaste previamente. Empieza a escribr tu codigo. Haz cambos en settings en tu repositorio y sube tu pagina.
